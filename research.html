RESEARCH

WHY RESEARCH?


Research has been a very important factor that has helped in determining the way my web pages looks to the way it interacts. For a beginner, it is very essential to take reference of other web pages developed by expierenced and professional web developer so that a concept gets developed.
The top 5 web pages I had taken references from, is listed as below:

1) Brainhub

This web page has been very helpful to me as I have designed my home page with reference to it. The big background and a overlay was created in reference to this page. I have also got an idea of creating a side navigation bar while going through this page. The thing I like about this website is it has minimal but effective design.

2) Crypton

From this webpage I had gained a lot knowledge on making web page interesting for user to view by the use of animations. Just like in this webpage I have used typing text animation in almost all of my pages. This website is kind of animative but the animation were not too disturbing to watch which is a good point of this website.

3) Wade jeffrey

This website is presented in a way that the viewer gets the personality of the developer. The home page has a video on loop where the designer is being hit by the basketball. The work of the designer leaves impression on viewer.

4) Marleigh Culver

This website is simple but simple takes more effort and the effort was seen. The color and the font was carefully chosen which gives complementry to the website. The navigation bar were easy to access and the fast loading of the image makes the website more viewer friendly.

5) Justin M. Maller

This website opens up in the work gallery. Here also the design was simple yet very creative. The fixed footer in this page caught my eyes.
